# Deck Build

## Generate the PPTX

```sh
node deck/build.js
```

This writes:

- `deck/should-you-trust-an-ai-mathematician.pptx`

## Notes

- The current deck is a first-pass skeleton with several intentional image placeholders.
- Asset status is tracked in `assets/manifest.md`.
- Typography and layout choices are intentionally conservative to keep later editing on iPad painless.
